## Gradle 简介
Java世界中主要有三大构建工具：Ant、Maven和Gradle。

Ant：软件编译、测试、部署等步骤联系在一起加以自动化的一个工具，但是依赖管理很差。（XML）
MAVEN：【依赖管理】，但是XML作为配置文件的格式有严格的结构层次和标准，定制化目标（goal）很困难。很难定制化构建脚本。
Gradle：有Ant的强大和灵活，又有Maven的生命周期管理且易于使用。
