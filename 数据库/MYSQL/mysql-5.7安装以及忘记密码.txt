1.配置系统变量：path为mysql安装目录下的bin目录；
2.mysql根目录下添加“my.ini”


[mysql]
# 设置mysql客户端默认字符集
default-character-set=utf8
[mysqld]
#设置3306端口
port = 3306
# 设置mysql的安装目录
basedir=D:\mysql5.7
# 设置mysql数据库的数据的存放目录
datadir=D:\mysql5.7\data
# 允许最大连接数
max_connections=200
# 服务端使用的字符集默认为8比特编码的latin1字符集
character-set-server=utf8
# 创建新表时将使用的默认存储引擎
default-storage-engine=INNODB

3.管理员运行cmd，执行以下目录

mysql安装
mysql install

mysqld --initialize --user=mysql --console
记住密码

mysqld --install MySQL

net start MySQL



----------------------忘记密码----------------------
1.关闭mysql服务。
2.在“my.ini”末尾添加“skip-grant-tables”
3.启动myqsl服务。
4.cmd进入到mysql。
5.更新密码。
update mysql.user set authentication_string=password('123456') where user='root' and Host = 'localhost';
6.flush privileges; quit;
8.关闭mysql服务。
9.在“my.ini”末尾删除“skip-grant-tables”
10.启动myqsl服务。

提示密码过期，cmd登陆后执行“SET PASSWORD = PASSWORD('123456');”
